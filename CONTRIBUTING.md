
## How To Contribute

1. Add a new `.json` file inside `cards/`. You can name the file anything you'd want plus your name at the end and a number.
2. Add that filename to `cards/index.json`.
3. Use the same shape for every file:

```json
{
  "title": "Your Card Title",
  "text": "Your card copy goes here.",
  "theme": "One of the theme options below."
}
```

4. Reload the website after the change is merged.

Example `cards/index.json`:

```json
[
  "example-halloween.json",
  "example-ocean.json",
  "example-spring.json",
  "example-starlight.json",
  "example-summer.json",
  "example-y2k.json"
]
```
