# Hello Wall

A beginner-friendly open source project used to practice the core Git workflow: creating branches, committing changes, and opening pull requests. Participants add themed greeting cards to a shared collection and see their contributions live on the "wall."

This project is demoed during workshops to give contributors a low-stakes, visual way to experience the full GitHub contribution cycle.

## About

Hello Wall is a simple, static site that renders themed greeting cards from JSON files. It is intentionally minimal so that contributors can focus on the Git workflow rather than the code.

The project is intended to be forked and used as a workshop starter. As such, it will implement:

- A card registry driven by `cards/index.json`
- A set of predefined visual themes contributors can choose from
- Live re-rendering when new cards are merged into `main`

## Getting Started

The live site is hosted on GitHub Pages: **https://opensource-stanford.github.io/hello-wall/**

No local setup is required to contribute a card. All you need is a GitHub account.

If you'd like to run the site locally:

1. Clone the repository
2. Open `index.html` in your browser — or run a local dev server of your choice
3. Changes to `cards/` will reflect immediately on reload
